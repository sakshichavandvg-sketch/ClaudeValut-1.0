# ClaudeVault

ClaudeVault is a modern full-stack web application.

## Project Structure

- `backend/`: FastAPI application containing the backend API.
- `claudevault-react/`: React application containing the frontend.

## Getting Started

### Prerequisites
- Python 3.9+
- Node.js & npm

### Backend Setup
1. Navigate to the `backend` directory (or use the root `requirements.txt`).
2. Create and activate a virtual environment.
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the FastAPI server.

### Frontend Setup
1. Navigate to the `claudevault-react` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm start
   ```
